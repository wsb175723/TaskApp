# Task Management App

Projekt aplikacji typu **Task Management App** zrealizowany w ramach przedmiotu **Projektowanie Interfejsów Użytkownika**.

Aplikacja została zaprojektowana jako jednostronicowa aplikacja internetowa (SPA), umożliwiająca wygodne zarządzanie zadaniami. Główny nacisk położono na prostotę obsługi, czytelny interfejs oraz responsywność.

## Kluczowe funkcjonalności

- Pełna obsługa operacji CRUD (dodawanie, odczyt, edycja i usuwanie zadań)
- Zarządzanie kategoriami
- Ustawianie priorytetu zadań
- Sortowanie, filtrowanie oraz wyszukiwanie zadań
- Obsługa pojedynczych terminów oraz przedziałów czasowych
- Widok kalendarza
- Statystyki zadań
- Tryb jasny i ciemny (Dark Mode)
- Powiadomienia typu Toast
- Eksport danych w formacie JSON
- Zapisywanie danych w localStorage
- Responsywny interfejs użytkownika

## Technologie

- React 18 (CDN)
- JavaScript (ES6)
- HTML5
- CSS3
- Tailwind CSS
- Web Storage API (localStorage)

## Jak uruchomić?

Aplikacja nie wymaga instalacji dodatkowych bibliotek ani środowiska.

1. Pobierz repozytorium.
2. Otwórz plik `projekt.html` w dowolnej przeglądarce internetowej (Chrome, Firefox, Edge).
